# @brand-icons/react

Year-aware, tree-shakable brand-icon components for React 18+.

## Install

```sh
pnpm add @brand-icons/react
# or
npm install @brand-icons/react
```

`react` and `react-dom` are peer dependencies (`>=18`).

## Usage

Flat naming: one component per millésime + a `Latest` alias.

```tsx
import { Apple1976Icon, Apple2017Icon, AppleLatestIcon } from '@brand-icons/react';

export const Page = () => (
  <>
    <Apple1976Icon size={32} />     {/* rainbow original */}
    <Apple2017Icon size={32} />     {/* current flat silhouette */}
    <AppleLatestIcon size={32} />   {/* alias = Apple2017Icon */}
  </>
);
```

For maximum tree-shaking, import the per-icon path directly — only the
single SVG is bundled:

```tsx
import { Apple1976Icon } from '@brand-icons/react/icons/Apple1976';
```

## Props

```ts
type BrandIconProps = {
  size?: number | string;        // default '1em'
  color?: string;                // only honored when mode='mono'
  variant?: 'color' | 'mono';    // default 'color'
  mode?: 'as-is' | 'bw' | 'wb' | 'mono'; // default 'as-is'
  background?: boolean | string; // true → brandColor, string → any CSS color
  title?: string;                // adds <title> + role="img" + aria-label
  className?: string;
};
```

### ColorMode

Runtime visual transform — no pre-generated derivative variants.

| Mode    | Variant rendered | CSS applied                              |
| ------- | ---------------- | ---------------------------------------- |
| `as-is` | honors `variant` | none                                     |
| `bw`    | honors `variant` | `filter: grayscale(1)`                   |
| `wb`    | honors `variant` | `filter: grayscale(1) invert(1)`         |
| `mono`  | forced to `mono` | `color: <color>` (defaults `currentColor`) |

```tsx
<GitHub2013Icon mode="bw" />              {/* grayscale */}
<GitHub2013Icon mode="wb" />              {/* white-on-black */}
<GitHub2013Icon mode="mono" color="#f00" /> {/* red mono silhouette */}
```

### Background

Wraps the icon in a `<rect>` painted with the brand's official color (or
a custom one).

```tsx
<Apple2017Icon size={48} background />          {/* black bg (brandColor) */}
<Apple2017Icon size={48} background="#ff0000" /> {/* explicit red bg */}
```

## Generated — do not edit

Every file under `src/icons/` is generated by `@brand-icons/build-icons`
from the source SVGs in `icons/<slug>/<year>/`. Edits live there; this
package is a thin React adapter rendering the inlined SVG strings from
`@brand-icons/core`.

See the project [docs site](https://github.com/bryanbergerprojects/brand-icons)
for the full gallery and authoring guide.
