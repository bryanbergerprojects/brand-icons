# Brand Icons — Documentation site

Next.js 15 site that documents the Brand Icons library and provides:

- A searchable icon gallery.
- A live playground per icon (size, variant, background, color).
- Per-framework code snippets and SVG downloads.
- MDX documentation pages.
- A page for brand owners (takedown procedure).

## Stack

- Next.js 15 App Router
- Tailwind CSS v4 (CSS-first config via `@theme`)
- shadcn/ui v4
- lucide-react (UI icons — not the catalogue)
- `@next/mdx`, Shiki, Fuse.js, next-themes
