## Install

Brand-icons ships per framework. Pick the package that matches your stack.

```bash
pnpm add @brand-icons/react
# or
pnpm add @brand-icons/vue
pnpm add @brand-icons/svelte
pnpm add @brand-icons/wc
```

Every framework package depends on `@brand-icons/core`, which carries the
manifest, SVG strings, and TypeScript types. Install it explicitly only if you
need the metadata in your own code:

```bash
pnpm add @brand-icons/core
```

## Render an icon

Each brand exposes a `<Brand>LatestIcon` component plus one component per
known millésime (`<Brand><Year>Icon`).

```tsx
import { GitHubLatestIcon } from '@brand-icons/react';

export const Example = () => <GitHubLatestIcon size={32} />;
```

## Variants

Switch between `color` and `mono` with the `variant` prop:

```tsx
<GitHubLatestIcon variant="mono" size={24} />
```

Monochrome icons use `currentColor`, so they inherit the parent text color.
