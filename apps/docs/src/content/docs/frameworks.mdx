## React

```tsx
import { AppleLatestIcon } from '@brand-icons/react';

<AppleLatestIcon size={48} variant="color" />
```

## Vue

```vue
<script setup>
import { AppleLatestIcon } from '@brand-icons/vue';
</script>

<template>
  <AppleLatestIcon :size="48" variant="color" />
</template>
```

## Svelte

```svelte
<script>
  import { AppleLatestIcon } from '@brand-icons/svelte';
</script>

<AppleLatestIcon size={48} variant="color" />
```

## Web Components

```html
<script type="module">
  import '@brand-icons/wc';
</script>

<brand-icon-apple-latest size="48" variant="color"></brand-icon-apple-latest>
```

## Shared prop API

Every framework exposes the same props (typed via `BrandIconProps` in
`@brand-icons/core`):

- `size` — number or string, default `24`
- `variant` — `'color' | 'mono'`, default `'color'`
- `mode` — `'as-is' | 'bw' | 'wb' | 'mono'`, default `'as-is'`
- `color` — CSS color, applies in `mono` variant or when mode forces a swap
- `background` — boolean (uses brand color) or string (custom hex)
- `title` — accessible label; sets `role="img"` and a `<title>`
- `className` — passthrough class on the root `<svg>`
